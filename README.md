# Álbum Pessoal — Vite + Tailwind (estático, compatível com GitHub Pages)

Este projeto é um álbum de fotos pessoal 100% frontend. Funcionalidades principais:
- Upload de fotos pelo navegador.
- Armazenamento local das fotos e metadados no IndexedDB do navegador.
- Visual com aparência de álbum físico: polaroids, fitas, notas manuscritas e animação de virar página.
